DOCUMENTAÇÂO Angular
COMANDOS:
https://angular.io/ -> DOCS -> References -> CLI Command References -> Overview -> onde existe todos os comandos

Instalar o Angular -> npm install -g @angular/cli

Criar o projeto -> ng new curso-angular

Startar a aplicação -> ng serve ou ng s

Instalar o Angular Material -> ng add @angular/material

Criando o Módulo de Cursos com roteamento -> ng g m courses --routing

Criando um componente dentro do Módulo -> ng g c courses/courses

Criando uma interface -> ng g interface courses/model/course
